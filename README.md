# Estudo de Map, Reduce e Filter em JavaScript

Projeto de estudo do canal **DevClub**, mostrando exemplos práticos de **Map**, **Reduce** e **Filter** em arrays.

 **Map**: criar um novo array modificando os itens de um array existente. **(Mapear item por item do Array)**

  - **Map tem acesso a 3 dados:**
    - Item por Item do array
    - Posição atual do Array
    - Array Completo  
    
 **Reduce**: calcular valores acumulados a partir dos itens de um array. **(Reduzir o Array a um único valor)**

 - **Reduce tem acesso a 4 dados:**
    - Acumulador
    - Valor Atual
    - Posição atual
    - Array Completo  
    
 **Filter**: Cria um novo array filtrando os valores desejados do array Original. **(Filtrar o array)**

  - **Filter tem acesso a 3 dados:**
    - Item por Item do array
    - Posição atual do Array
    - Array Completo
